# Node.js 基本内置模块

---

## 一、os 模块

`os`（Operating System）模块提供了与操作系统相关的实用方法和属性。

| API | 说明 | 返回值示例 |
|---|---|---|
| `os.EOL` | 当前操作系统的行结束标记 | Windows: `\r\n`，Linux/Mac: `\n` |
| `os.arch()` | 操作系统 CPU 架构 | `x64` |
| `os.cpus()` | 返回每个逻辑 CPU 内核的信息数组 | 对象数组（包含 model、speed 等） |
| `os.freemem()` | 系统当前空闲内存（字节） | 数字 |
| `os.homedir()` | 当前用户主目录路径 | `C:\Users\EDY` |
| `os.hostname()` | 操作系统主机名 | `WINDOWS-5CVHPFE` |
| `os.tmpdir()` | 系统默认临时文件夹路径 | `C:\Users\EDY\AppData\Local\Temp` |

### 示例

```js
const os = require("os");

// 系统架构
console.log(os.arch()); // x64

// CPU 核心数
console.log(os.cpus().length);

// 空闲内存（换算为 GB）
console.log(os.freemem() / 2 ** 30);

// 用户主目录
console.log(os.homedir()); // C:\Users\EDY

// 主机名
console.log(os.hostname()); // WINDOWS-5CVHPFE

// 临时文件夹路径
console.log(os.tmpdir()); // C:\Users\EDY\AppData\Local\Temp
```

## 二、path 模块

`path` 模块用于处理和转换文件路径。

| API | 说明 |
|---|---|
| `path.basename(p, ext?)` | 返回路径的最后一部分（文件名），可选参数 `ext` 用于移除扩展名 |
| `path.sep` | 当前系统的路径分隔符 |
| `path.delimiter` | 当前系统的环境变量分隔符 |
| `path.dirname(p)` | 返回路径的目录部分 |
| `path.extname(p)` | 返回路径的扩展名（含 `.`） |
| `path.join(...paths)` | 拼接路径片段，自动处理 `..` 和 `.` |
| `path.relative(from, to)` | 返回从 `from` 到 `to` 的相对路径 |
| `path.resolve(...paths)` | 将路径片段解析为绝对路径 |

### 示例

```js
const path = require("path");

// basename —— 获取文件名（第二个参数可指定要去除的后缀）
const basename = path.basename("fdg/dfgdfg/adfaf/fdgdfgd/a.asf", ".html");
console.log(basename); // a.asf（后缀不匹配 .html，所以不去除）

// 当前系统的路径分隔符
console.log(path.sep); // windows: \  |  linux: /

// 环境变量分隔符
console.log(path.delimiter); // windows: ;  |  linux: :
console.log(process.env.PATH.split(path.delimiter));

// dirname —— 获取目录部分
const dir = path.dirname("a/b/c/d");
console.log(dir); // a/b/c

// extname —— 获取扩展名
const ext = path.extname("a/b/c/a.js");
console.log(ext); // .js

// join —— 拼接路径（支持 .. / . 等相对路径）
const basePath = "a/b";
const fullpath = path.join(basePath, "../", "d.js");
console.log(fullpath); // a\d.js

// relative —— 获取相对路径
const rel = path.relative("/data/orandea/test/aaa", "/data/orandea/impl/bbb");
console.log(rel); // ..\..\impl\bbb

// resolve —— 解析为绝对路径
const absPath = path.resolve(__dirname, "./a.js");
console.log(absPath);
```

## 三、url 模块

`url` 模块用于 URL 解析与格式化。

| API | 说明 |
|---|---|
| `new URL(input)` | 解析 URL 字符串为 URL 对象 |
| `url.searchParams.has(key)` | 查询参数是否存在 |
| `url.searchParams.get(key)` | 获取指定查询参数的值 |
| `url.format(obj)` | 将 URL 对象格式化为字符串 |

### 示例

```js
const url = require("url");

const url1 = new url.URL("https://nodejs.org:80/a/b/c?t=3&u=5#abc");

console.log(url1);

// searchParams 操作
console.log(url1.searchParams.has("a")); // false
console.log(url1.searchParams.has("t")); // true
console.log(url1.searchParams.get("t")); // "3"

// format —— 将对象序列化为 URL 字符串
const obj = {
  href: "https://nodejs.org:80/a/b/c?t=3&u=5#abc",
  origin: "https://nodejs.org:80",
  protocol: "https:",
  username: "",
  password: "",
  host: "nodejs.org:80",
  hostname: "nodejs.org",
  port: "80",
  pathname: "/a/b/c",
  search: "?t=3&u=5",
  hash: "#abc",
};

const url2 = url.format(obj);
console.log(url2); // https://nodejs.org:80/a/b/c?t=3&u=5#abc
```

## 四、util 模块

`util` 模块提供了一些实用工具函数。

| API | 说明 |
|---|---|
| `util.callbackify(fn)` | 将返回 Promise 的异步函数转为回调风格 |
| `util.promisify(fn)` | 将回调风格的函数转为返回 Promise |
| `util.isDeepStrictEqual(a, b)` | 深度比较两个值是否严格相等 |

### 示例

```js
const util = require("util");

// 1. callbackify —— Promise → 回调
async function delay(duration = 1000) {
  return new Promise(resolve => {
    setTimeout(() => {
      resolve(duration);
    }, duration);
  });
}

const delayCallback = util.callbackify(delay);

delayCallback(500, (err, d) => {
  console.log(d); // 500
});

// 2. promisify —— 回调 → Promise
function delayCallBack(duration, callback) {
  setTimeout(() => {
    callback(null, duration);
  }, duration);
}

const delayPromise = util.promisify(delayCallBack);

(async () => {
  const r = await delayPromise(500);
  console.log(r); // 500
})();

// 3. isDeepStrictEqual —— 深度比较
const obj1 = {
  a: 1,
  b: {
    c: 3,
    d: { e: 5 },
  },
};

const obj2 = {
  a: 1,
  b: {
    c: 3,
    d: { e: 5, g: 6 },
  },
};

console.log(util.isDeepStrictEqual(obj1, obj2)); // false
```
