## 1. global 对象

`global` 是 Node.js 中的全局命名空间对象。我们可以向其挂载自定义数据，以便在全局访问。

```javascript
// 给 global 挂载数据
global.User = {
  name: "Yan",
  age: 18
};

console.log(global.User); // { name: 'Yan', age: 18 }
```

## 2. 模块级全局变量

在 CommonJS 中，`__filename` 和 `__dirname` 是默认提供的。但在 ES Module 模式下，需要通过 `url` 和 `path` 模块手动转换获取。

- `__filename`: 当前模块的文件绝对路径。
- `__dirname`: 当前模块所在的目录绝对路径。

```javascript
import { fileURLToPath } from 'url';
import { dirname } from 'path';

const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);

console.log(__filename); // 例如：e:\p-workspace\Test\node\1.global.js
console.log(__dirname);  // 例如：e:\p-workspace\Test\node
```

## 3. process 进程对象

`process` 对象提供了当前 Node.js 进程的相关信息和控制方法。

### 获取进程信息

```javascript
console.log(process.argv);     // 获取执行时携带的 CLI 参数数组
console.log(process.cwd());    // 获取当前 Node 进程执行时的工作目录
console.log(process.env);      // 获取操作系统环境变量对象
console.log(process.version);  // 获取 Node.js 版本号
console.log(process.pid);      // 获取当前进程的 PID (进程 ID)
console.log(process.platform); // 获取运行环境的操作系统平台 (如 'win32')
console.log(process.arch);     // 获取 CPU 架构信息 (如 'x64')
```

### 退出进程

```javascript   
process.exit(1); // 强制退出当前进程，传入 1 表示告诉操作系统非正常退出（0 表示正常退出）
process.kill(30259); //强制杀死进程,传入进程id
```

### 标准输入输出流

- **`process.stdout`**: 标准输出流，可用于在控制台输出内容。相比 `console.log`，`stdout.write` 不会自动换行。
- **`process.stdin`**: 标准输入流，用于监听和获取用户在控制台的输入。

```javascript
process.stdout.write('hello '); 
process.stdout.write('world'); // 输出为: hello world
```

## 4. Buffer 二进制数据处理

`Buffer` 类用于在 Node.js 中处理二进制数据流。

### 创建 Buffer

```javascript
// 创建一个 10 字节的 Buffer，并用 0 填充
const buf1 = Buffer.alloc(10); 

// 通过字符串创建
const buf2 = Buffer.from('Hello, world!'); 

// 通过十六进制数组创建
const buf3 = Buffer.from([0x68, 0x65, 0x6c, 0x6c, 0x6f]); 
```

### 格式转换

```javascript
console.log(buf2.toString());          // 转换为普通字符串: 'Hello, world!'
console.log(buf3.toString('base64'));  // 转换为 base64 编码格式: 'hello' 的 base64 形式
console.log(Array.from(buf2));         // 将 Buffer 对象转换为字节数组
```

### 写入内容

```javascript
const buff = Buffer.alloc(11);
buff.write('Hello ');        // 写入 Buffer 的前部分字节
buff.write(' world', 5);     // 从索引 5 开始写入，不会覆盖之前的内容
console.log(buff.toString());// 输出: 'Hello world'
```

### 合并与截取 Buffer

```javascript
// 合并多个 Buffer
const b1 = Buffer.from('Hello');
const b2 = Buffer.from('World');
const b3 = Buffer.concat([b1, b2]);
console.log(b3.toString()); // 'HelloWorld'

// 截取 Buffer (浅拷贝)
const b4 = Buffer.from('Hi,there');
const b5 = b4.subarray(0, 2); // 截取索引 0 到 2 (不包含 2) 的内容
console.log(b5.toString());   // 'Hi'

// 拷贝出指定索引的独立 Buffer (深拷贝)
const b6 = Buffer.from(b5, 0, 2); 
```
