# nodemon使用
nodemon是一个基于nodejs的实时重启工具，当代码发生改变时，会自动重启服务。

## 安装
```sh
npm i nodemon -S
```

## 使用
```sh
nodemon app.js
```

package.json配置

```json
"scripts": {
    "start": "nodemon -x npm run server",//-x 表示执行后面的命令
    "server": "node index.js"
}
```

根目录新建`nodemon.json`可以对nodemon进行配置
```json
{
  "env": {
    "NODE_ENV": "development"
  },
  "watch": ["*.js", "*.json"],
  "ignore": ["package*.json", "nodemon.json", "node_modules", "public"]
}
```

```js
console.log(process.env.NODE_ENV);//获取当前环境变量
```
