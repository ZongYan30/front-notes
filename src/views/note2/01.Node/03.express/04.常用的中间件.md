# 常用内置中间件
在之前原始的http模块中,前端发送的资源都是以流的形式发送的,需要我们手动的解析。
express提供了一些内置的中间件，用于处理一些常见的问题。

* express.static()
* express.json()
* express.urlencoded()

## 表单解析处理
对应前端发送的`www-form-urlencoded`格式

```js
const express = require("express");
const app = express();

app.use(express.urlencoded({ extended: true }));//解析表单格式

/* 路由 */
app.post("/api/student", (req, res) => {
  //如果不使用urlencoded中间件,req.body为undefined
  console.log(req.body);//{name:'张三',age:18}
});

/* 错误处理中间件 */
app.use(require("./errorMiddleware"));

app.listen(5008, () => {
  console.log(`server listen on ${5008}`);
});

```

## json处理
对应前端发送的`json`格式

const express = require("express");

## 静态资源处理

```js
const express = require("express");
const app = express();
const path = require("path");

/* 静态资源中间件 */
/**
 * 下面这段代码的作用：
 * 当请求时，会根据请求路径(req.path)，从指定的目录中寻找是否存在该文件，如果存在，直* 接响应文件内容，而不再移交给后续的中间件
 * 如果不存在文件，则直接移交给后续的中间件处理
 * 默认情况下，如果映射的结果是一个目录，则会自动使用index.html文件
 */

const staticRoot = path.resolve(__dirname, "../public");
app.use(express.static(staticRoot));


app.post("/api/student", (req, res) => {
  console.log(req.body);//{name:'张三',age:18}
});

app.listen(5008, () => {
  console.log(`server listen on ${5008}`);
});
```

## 最佳实践
中间件是有使用顺序的
* 静态资源中间件(先判定是否是静态资源,是的话直接就响应,不会继续执行后续的中间件)
* 表单解析中间件(解析表单格式的请求体)
* json解析中间件(解析json格式的请求体)
* 路由中间件(处理路由请求)
* 错误处理中间件(处理错误)

```js
const express = require("express");
const app = express();
const path = require("path");

// 静态资源
// app.use(express.static(staticRoot));

app.use("/static", (req, res) => {
  console.log(req.baseUrl, req.path);
});

// 表单解析
app.use(express.urlencoded({ extended: true }));

// json解析
app.use(express.json());
// app.use(require("./myUrlEncoded"));

// 路由
app.post("/api/student", (req, res) => {
  console.log(req.body);
});

// 错误处理
app.use(require("./errorMiddleware"));

const port = 5008;
app.listen(port, () => {
  console.log(`server listen on ${port}`);
});
```

## 拓展-实现urlencoded中间件
```js
const qs = require("querystring");

module.exports = (req, res, next) => {
  if (req.headers["content-type"] === "application/x-www-form-urlencoded") {
    //自行解析消息体
    let result = "";

    req.on("data", (chunk) => {
      result += chunk.toString("utf-8");
    });

    req.on("end", () => {
      const query = qs.parse(result);
      req.body = query;
      next();
    });
    
  } else {
    next();
  }
};
```
