# mysql驱动程序

**什么是驱动程序**
驱动程序是连接内存和其他存储介质的桥梁,mysql驱动程序是连接内存数据和mysql数据的桥梁。
mysql驱动程序通常使用:

- mysql
- mysql2 (推荐,和官方合作，更稳定)

## mysql2的使用

```sh
npm install  mysql2 --save
```

:::code-group

```js [回调风格.js]
const mysql = require("mysql2");

// 创建一个数据库连接
const connection = mysql.createConnection({
  host: "localhost",
  user: "root",
  password: "root",
  database: "companydb",
});

// 查询
connection.query("SELECT * FROM `company`;", function (err, results) {
  //err错误
  //result查询结果
  console.log(results); // results contains rows returned by server
});

// 插入
connection.query(
  "insert into company(`name`,location,buildDate) values('abbc', '阿萨德', curdate());",
  (err, result) => {
    console.log(result);
  },
);

// 修改
connection.query(
  "update company set `name`='bcd' where id=4",
  (err, result) => {
    console.log(result);
  },
);

// 删除
connection.query("delete from company where id=4", (err, result) => {
  console.log(result);
});
```

```js [promise风格.js]
const mysql = require("mysql2/promise");

async function test() {
  const connection = await mysql.createConnection({
    host: "localhost",
    user: "root",
    password: "root",
    database: "companydb",
  });

  const [results] = await connection.query("SELECT * FROM `company`;");
  console.log(results);
  connection.end(); //结束这次连接
}

test();
```

:::

## sql注入

用户通过注入sql语句到最终查询中，导致了整个sql与预期行为不符。

```js
test(id){
    const sql=`select * from company where id=${id}`;
    await connection.query(sql);
}

const injectSql=`"";DELETE FROM company where id=4;`;
test(id);

// 注入的语句变成了:
// select * from company where id="";DELETE FROM company where id=4;
```

原本是做查询操作，最终被恶意注入变成了删除操作。

## 最佳实践

mysql支持变量,变量的内容不作为任何sql关键字。

针对sql注入，mysql2提供了预编译API,将语句预编译直接定着执行的格式，之后再绑定参数就不会产生注入了。

```js
const mysql = require("mysql2/promise");

// 这里创建成一个连接池，
const pool = mysql.createPool({
  host: "localhost",
  user: "root",
  password: "root",
  database: "companydb",
  multipleStatements: true, //开启多语句执行
  waitForConnections: 10, //等待连接数
  connectionLimit: 10, //最大连接数
  queueLimit: 0, //等待可排队数，0表示不限制
});

async function test(id) {
  //在模板字符串中,模糊查询需要使用concat连接%号
  const sql = `select * from employee where \`name\` like concat('%', ?, '%') ;`;
  const [results] = await pool.execute(sql, [id]); //之后都推荐使用execute方法
  console.log(results);
}

test("袁");
```
