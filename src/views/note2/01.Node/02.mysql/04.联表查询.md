# 联表查询

## 笛卡尔积

**笛卡尔积**是最基础的连接，它将左表的每一行与右表的每一行组合，结果集行数 = 左表行数 × 右表行数。

```sql
-- CROSS JOIN
SELECT * FROM table1 CROSS JOIN table2;
-- 隐式连接
SELECT * FROM table1, table2;
```

## 内连接

内连接是最常用的连接，只返回两个表中匹配的行。(A ∩ B)

```sql
-- inner join
SELECT * FROM table1
INNER JOIN table2 ON table1.column = table2.column;
-- 简写
SELECT * FROM table1
JOIN table2 ON table1.column = table2.column;
```

## 左外连接

左连接返回左表的所有行，即使右表没有匹配。右表没有匹配的行，结果中右表字段为 NULL。

```sql
-- 左连接
SELECT * FROM table1
LEFT JOIN table2 ON table1.column = table2.column;

-- 查询部门下的所有员工
SELECT * from department as d
left join employee as e
on d.id = e.deptId;
```

## 右外连接

右连接是左连接的反向，返回右表的所有行，即使左表没有匹配。(通常使用左连接比较多，右连接只用左连接交换顺序即可)

```sql
SELECT * FROM table1
RIGHT JOIN table2 ON table1.column = table2.column;

-- 查询员工及所属部门
SELECT * from employee as e
RIGHT JOIN department as d
on d.id = e.deptId;
```

## 练习

```sql
-- 1. 创建一张team表，记录足球队
-- 查询出对阵表
SELECT t1.name 主场, t2.name 客场
FROM team as t1, team as t2
WHERE t1.id != t2.id;

-- 2. 显示出所有员工的姓名、性别（使用男或女显示）、入职时间、薪水、所属部门（显示部门名称）、所属公司（显示公司名称）
SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id

-- 3. 查询腾讯和蚂蚁金服的所有员工姓名、性别、入职时间、部门名、公司名
SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id
WHERE c.`name` in ('腾讯科技', '蚂蚁金服')

-- 4. 查询渡一教学部的所有员工姓名、性别、入职时间、部门名、公司名
SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id
WHERE c.`name` like '%渡一%' AND d.`name` = '教学部';

-- 5. 列出所有公司员工居住的地址（要去掉重复）
select DISTINCT location from employee;
```
