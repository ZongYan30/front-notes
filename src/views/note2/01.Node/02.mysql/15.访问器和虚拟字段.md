## sequlize访问器
查看当前学生表发现,学生的生日是这样的格式:
```
2023-06-16T00:00:00.000Z
```

我们期望当获取的时候，可以读成时间戳的形式,sequlize在定义模型的时候提供了访问器getter和setter。

```js
// 创建一个模型对象
const Student = sequelize.define(
	"Student",
	{
        //...其他

		birthdady: {
			type: DataTypes.DATE,
			allowNull: false,
            //读取的时候转换为时间戳
			get() {
				return this.getDataValue("birthdady").getTime();
			}
		}

        //...其他
	},
	{
		createdAt: false,
		updatedAt: false,
		paranoid: true
	}
)
```

## 虚拟字段
虚拟字段就是定义在模型上的字段，但是数据库中并不存在，类似于vue中的计算属性`computed`。
```js
const sequelize = require("./db");
const { DataTypes } = require('sequelize');
const moment = require("moment");

// 创建一个模型对象
const Student = sequelize.define(
	"Student",
	{
        //...其他

		birthdady: {
			type: DataTypes.DATE,
			allowNull: false,
            //读取的时候转换为时间戳
			get() {
				return this.getDataValue("birthdady").getTime();
			}
		},

        //虚拟字段,从生日计算年龄
        age: {
			type: DataTypes.VIRTUAL,
			get() {
				const now = moment.utc();
				const birth = moment.utc(this.getDataValue("birthdady"));
				return now.diff(birth, "year");
			}
		},

        //...其他
	},
	{
		createdAt: false,
		updatedAt: false,
		paranoid: true
	}
)
```

最终,查询一下结果就变成了:
```js
{
	birthdady: 349401600000,
	age: 45,
	id: 1,
	name: '邹刚',
	sex: false,
	Class: null
}
```
