# 模拟数据

可以使用mock.js模拟生成一些数据，插入到数据库。

## 安装mock

使用[mockjs](http://mockjs.com)来模拟生成数据

```sh
npm install mockjs -S
```

## 使用mock

```sh
├─ models
│  ├─ relation.js 新增表关系文件
│  ├─ Class.js
│  └─ Student.js
├─ mock
│  ├─ mockStudents.js  模拟学生
│  └─ mockClasses.js   模拟班级
```

:::code-group

```js [mockClasses.js]
var Mock = require("mockjs");
const Class = require("../models/Class.js");

var result = Mock.mock({
  "datas|16": [
    {
      // 属性 id 是一个自增数，起始值为 1，每次增 1
      "id|+1": 1,
      name: "前端第 @id 期",
      openDate: "@date",
    },
  ],
}).datas;

console.log(JSON.stringify(result, null, 4));

//sequelize提供批量插入到数据库API，重复执行时按主键更新，避免主键冲突
Class.bulkCreate(result, {
  updateOnDuplicate: ["name", "openDate"],
});
```

```js [mockStudents.js]
const Mock = require("mockjs");
const Student = require("../models/Student");

const result = Mock.mock({
  "datas|500-700": [
    {
      name: "@cname",
      birthdady: "@date",
      "sex|1-2": true,
      mobile: /1\d{10}/,
      //   location: "@city(true)",
      "ClassId|1-16": 0,
    },
  ],
}).datas;

console.log(result);

Student.bulkCreate(result);
```

```js [models/relation.js]
// 表关系的映射会专门拎在一个文件中执行
const Class = require("./Class.js");
const Student = require("./Student.js");

Class.hasMany(Student); //表关系设置
```

:::

最终执行index入口文件,去数据库查看对应的数据即可

```js
require("./models/relation.js");
require("./mock/mockClass.js");
require("./mock/mockStudent.js");
```
