# 单表基本查询

声明式语句

```sql
select ... from  ... where ... order by ... limit ...
```

## select

| 关键字   | 作用     |
| -------- | -------- |
| as       | 别名     |
| `*`      | 所有字段 |
| case     | 条件语句 |
| distinct | 去重     |

```sql
-- as 别名
SELECT id, loginid, loginpwd, 'abc' as '额外的一列' from `user`;
SELECT ismale '性别' from `employee`;

-- 查询所有
SELECT * from `user`;
SELECT *, 'abc' as 'extra' from `employee`;

-- case语句
SELECT id,
`name`,
case ismalewhen 1 then '男'else '女'end sex,
salary
FROM employee;

SELECT id, `name`,
case
when ismale = 1 then '男'
else '女'
end sex,
case
when salary>=10000 then '高'
when salary>=5000 then '中'
else '低'
end `level`,
salary
FROM employee;

-- DISTINCT 去重(一般只针对单个字段)
select DISTINCT location from employee;
```

## where

| 关键字/运算符     | 作用                                               |
| ----------------- | -------------------------------------------------- |
| `=`               | 等于                                               |
| `in`              | 在指定集合中                                       |
| `is`              | 判断是否为某个值，常用于 `null`、`true`、`false`   |
| `is not`          | 判断是否不为某个值，常用于 `null`、`true`、`false` |
| `>` `<` `>=` `<=` | 大于、小于、大于等于、小于等于                     |
| `between`         | 在某个区间范围内                                   |
| `like`            | 模糊匹配                                           |
| `and`             | 并且，多个条件同时成立                             |
| `or`              | 或者，多个条件满足其一                             |

```sql
-- where
SELECT * FROM employee WHERE ismale = 1;

-- in
SELECT * FROM department WHERE companyId in (1, 2);

-- is
SELECT * from employee WHERE location is not null;

SELECT * from employee WHERE location is null;

-- > < >= <=
SELECT * from employee WHERE salary>=10000;

-- between and
SELECT * from employee WHERE salary BETWEEN 10000 and 12000;

-- 模糊搜索 %-代表任意字符，_代表任意单字符
SELECT * from employee WHERE `name` like '%袁%';

SELECT * from employee WHERE `name` like '袁_';

-- 综合
SELECT * from employee
WHERE `name` like '张%' and ismale=0 and salary>=12000;

SELECT * from employee
WHERE `name` like '张%' and (ismale=0 and salary>=12000
or
birthday>='1996-1-1');
```

## order by

```sql
SELECT * from employee ORDER BY salary desc;

-- 综合使用(按照性别升序，再按照薪资降序)
SELECT *, case ismale
when 1 then '男'
else '女'
end sex from employee
ORDER BY sex asc, salary desc;
```

## limit

`limit 偏移量, 条数`，公式一般为: `limit (page-1)\*pagesize, pagesize`

```sql
-- 跳过前2条数据，查询3条数据
SELECT * from employee limit 2,3;
```

## 练习

```sql
-- 查询user表，得到账号为admin，密码为123456的用户
SELECT * FROM `user` WHERE `loginId`='admin' and `loginpwd`='123456';

-- 查询员工表，按照员工的入职时间降序排序，并且使用分页查询 查询第3页，每页5条数据
SELECT * FROM `employee` ORDER BY joinDate desc limit 10,5;

-- 查询工资最高的女员工(查出女的，按薪资降序，取第一个)
SELECT * FROM `employee` WHERE ismale=0 ORDER BY salary desc limit 1;
```

## 语句执行优先级

from -> where -> select --> order by -> limit
