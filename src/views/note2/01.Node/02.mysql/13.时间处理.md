# 时间处理
**1. utc和北京时间**

   utc：世界协调时，以英国格林威治时间为标准

   utc时间和北京时间相差8小时，utc的凌晨相当于北京时间的上午8点


**2. 时间戳 timestamp**

某个utc时间到utc1970-1-1凌晨经过的毫秒数,某个utc时间到utc1970-1-1凌晨经过的毫秒数

时间戳表示的是utc时间的差异


**3. 对于服务器的影响**

服务器可能会部署到世界的任何位置，服务器内部应该统一使用utc时间或时间戳，包括数据库


**4. 对于客户端的影响**

客户端要给不同地区的客户友好的显示时间，客户端应该把时间戳或utc时间转换为本地时间显示。

## 2. 获取当前时间

代码：

```js
moment()//获取当前本地时间戳，单位是毫秒
moment.utc()//获取当前 UTC 时间戳
```
两者返回的都是 Moment 对象，可以继续调用格式化、时间计算等方法。

---

## 3. 获取时间戳

代码：

```js
moment().valueOf()//得到当前时间对应的时间戳，单位是毫秒
+moment()//简写，因为+运算符内部自动调用valueOf()方法
```

---

## 4. 创建时间对象

Moment 可以根据不同类型的数据创建时间对象，例如时间戳、字符串等。

示例代码：

```js
//根据时间戳 `0` 创建本地时间对象
console.log(moment(0).toString(), +moment(0));
//根据时间戳 `0` 创建 UTC 时间对象
console.log(moment.utc(0).toString(), +moment.utc(0));
//根据字符串创建本地时间对象
const value = "1970-01-01 00:00:00";
console.log(moment(value).toString(), +moment(value));
//根据字符串创建 UTC 时间对象
console.log(moment.utc(value).toString(), +moment.utc(value));
```

---

## 5. 解析时间

```js
/* 
- `formats` 是一个格式数组，表示允许的输入格式。
- `YYYY-MM-DD HH:mm:ss`：标准年月日时分秒格式。
- `YYYY-M-D H:m:s`：允许个位数月、日、时、分、秒。
- `x`：表示毫秒时间戳。
*/
const formats = ["YYYY-MM-DD HH:mm:ss", "YYYY-M-D H:m:s", "x"];

console.log(moment.utc("1970-01-01 00:00:00", formats, true).toString());
console.log(moment.utc("1970-1-1 0:0:0", formats, true).toString());
console.log(moment.utc(0, formats, true).toString());
```

这里的参数含义：

- 第 1 个参数：要解析的时间值。
- 第 2 个参数：允许匹配的格式列表。
- 第 3 个参数 `true`：表示严格匹配。

---

## 6. 合法性判断

```js
//生成一个不合法的时间对象
const invalidMoment = moment.utc(
  "Thu Jan 01 1970 00:00:00 GMT+0000",
  formats,
  true
);

console.log(invalidMoment.isValid()); // false
console.log(invalidMoment.toString());//显示无效时间的字符串表示
console.log(+invalidMoment);//显示无效时间的时间戳，单位是毫秒
```

## 7. UTC 时间与本地时间互相转化
**1. 转本地**

- `local()` 可以把 UTC 时间转换为当前系统的本地时间。
- `format(...)` 用指定格式输出时间字符串。

```js
const m = moment.utc("2015-1-5 23:00:01", formats, true);
console.log(m.local().format("YYYY年MM月DD日 HH点mm分ss秒"));
```

常用于：

- 服务器存 UTC 时间。
- 前端或客户端展示时转换成本地时间。


**2. 转 UTC**
- `utc()` 把本地时间转换成 UTC 时间。

```js
const m = moment("2015-1-5 23:00:01", formats, true);
const toServer = m.utc().format("YYYY-MM-DD HH:mm:ss");
console.log(toServer);
```

常用于：

- 用户在本地输入时间。
- 前端统一转成 UTC 再提交给后端保存。

---

## 9. 获取相对时间

- `fromNow()` 用于显示“相对于现在”的时间描述。

```js
const m = moment.utc("2020-4-14 9:00:01", formats, true);
console.log(m.local().fromNow());//6年前
```

## 10. 本次代码中的 API 总结

| API | 用途 |
| --- | --- |
| `moment()` | 获取当前本地时间对象 |
| `moment.utc()` | 获取当前 UTC 时间对象，或按 UTC 方式解析时间 |
| `valueOf()` | 获取毫秒级时间戳 |
| `isValid()` | 判断时间是否有效 |
| `local()` | 将 UTC 时间转换为本地时间 |
| `utc()` | 将本地时间转换为 UTC 时间 |
| `format()` | 按指定格式输出时间字符串 |
| `fromNow()` | 以相对当前时间的方式显示结果 |
