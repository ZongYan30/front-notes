# 表的增删改

## 1. 创建表

| 类别     | 字段类型/属性  | 说明                                                 |
| -------- | -------------- | ---------------------------------------------------- |
| 字段类型 | `bit`          | 占 1 位，`0` 或 `1`，`false` 或 `true`               |
| 字段类型 | `int`          | 占 32 位，整数                                       |
| 字段类型 | `decimal(M,N)` | 能精确计算的实数，`M` 是总的数字位数，`N` 是小数位数 |
| 字段类型 | `char(n)`      | 固定长度为 `n` 的字符                                |
| 字段类型 | `varchar(n)`   | 长度可变，最大长度为 `n` 的字符                      |
| 字段类型 | `text`         | 大量的字符                                           |
| 字段类型 | `date`         | 仅日期                                               |
| 字段类型 | `datetime`     | 日期和时间                                           |
| 字段类型 | `time`         | 仅时间                                               |
| 字段属性 | 自增           | 是否自增                                             |
| 字段属性 | `null`         | 是否可以为 `null`                                    |
| 字段属性 | 默认值         | 字段默认值                                           |

```sql
CREATE TABLE `students` (
  `id` INT PRIMARY KEY AUTO_INCREMENT COMMENT '学号，主键',
  `name` VARCHAR(50) NOT NULL COMMENT '姓名',
  `age` TINYINT(1) COMMENT '年龄',
  `gender` VARCHAR(10) DEFAULT '未知' COMMENT '性别',
  `score` DECIMAL(5,2) COMMENT '分数'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 2. 插入数据

插入一条数据

```sql
INSERT INTO `students` (`name`, `age`, `gender`, `score`)
VALUES ('张三', 20, '男', 88.5);
```

插入多条数据

```sql
INSERT INTO `students` (`name`, `age`, `gender`, `score`) VALUES
('李四', 22, '女', 92.0),
('王五', 19, '男', 75.5),
('赵六', 21, '女', 83.0);
```

省略字段名

```sql
-- 因为 id 是自增的，可以写 NULL 或 0，让数据库自动生成
INSERT INTO `students` VALUES (NULL, '孙七', 23, '男', 67.5);
```

## 3. 更新数据

更新单条数据

```sql
-- 把 id=1 的学生分数改为 95.5
UPDATE `students` SET `score` = 95.5 WHERE `id` = 1;
```

更新多条数据

```sql
-- 把所有女生的年龄加 1 岁
UPDATE `students` SET `age` = `age` + 1 WHERE `gender` = '女';
```

同时更新多个字段

```sql
-- 修改 id=2 的姓名和分数
UPDATE `students` SET `name` = '李思', `score` = 96.0 WHERE `id` = 2;
```

## 4. 删除数据

删除指定记录

```sql
-- 删除 id=5 的学生
DELETE FROM `students` WHERE `id` = 5;
```

按条件删除

```sql
-- 删除所有分数低于 60 的学生
DELETE FROM `students` WHERE `score` < 60;
```

清空整张表

```sql
-- 方式一：DELETE FROM（逐行删除，可以加 WHERE，速度较慢）
DELETE FROM `students`;

-- 方式二：TRUNCATE TABLE（先删表再重建，速度快，但不能加 WHERE）
TRUNCATE TABLE `students`;
```

## 表关系

主键: 根据设计原则，每张表都要有主键，主键必须**唯一,不能更改,无业务含义**
外键:用于产生表关系的列,外键列会连接到另一张表（或自己）的主键。

1. 一对一
   一个A对应一个B，一个B对应一个A
   把任意一张表的主键同时设置为外键

   > 例如：用户和用户信息

2. 一对多
   一个A对应多个B，一个B对应一个A，A和B是一对多，B和A是多对一
   在多一端的表上设置外键，对应到另一张表的主键

   > 例如：班级和学生，用户和文章

3. 多对多
   一个A对应多个B，一个B对应多个A
   需要新建一张关系表，关系表至少包含两个外键，分别对应到两张表

   > 例如：学生和老师

### 三大设计范式

1. 要求数据库表的每一列都是不可分割的原子数据项
2. 非主键列必须依赖于主键列
3. 非主键列必须直接依赖主键列
