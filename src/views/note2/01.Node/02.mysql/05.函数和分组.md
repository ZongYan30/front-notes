## 函数

mysql提供了一些内置函数，类似JS中的内置函数供开发者使用。

### 数学函数

常用于绝对值、取整、随机数和数值处理。

| 函数                     | 作用                             |
| ------------------------ | -------------------------------- |
| `ABS(x)`                 | 返回 `x` 的绝对值                |
| `CEILING(x)` / `CEIL(x)` | 向上取整                         |
| `FLOOR(x)`               | 向下取整                         |
| `MOD(x, y)`              | 返回 `x` 除以 `y` 的余数         |
| `PI()`                   | 返回圆周率                       |
| `RAND()`                 | 返回 0 到 1 之间的随机数         |
| `ROUND(x, y)`            | 将 `x` 保留 `y` 位小数并四舍五入 |
| `TRUNCATE(x, y)`         | 将 `x` 保留 `y` 位小数，直接截断 |

```sql
SELECT ABS(-1);

SELECT CEIL(1.4);

SELECT ROUND(3.1415926, 3);

-- 保留2位小数，直接砍掉后面
SELECT TRUNCATE(3.1415926,3);

SELECT TRUNCATE(salary,0)
FROM employee
```

### 字符串函数

常用于字符串拼接与去空格处理。

| 函数                              | 作用                         |
| --------------------------------- | ---------------------------- |
| `CONCAT(s1, s2, ..., sn)`         | 拼接多个字符串               |
| `CONCAT_WS(sep, s1, s2, ..., sn)` | 使用指定分隔符拼接多个字符串 |
| `TRIM(str)`                       | 去掉字符串两端空格           |
| `LTRIM(str)`                      | 去掉字符串左侧空格           |
| `RTRIM(str)`                      | 去掉字符串右侧空格           |

### 日期函数

常用于获取当前日期时间以及计算时间差。

| 函数                                | 作用                             |
| ----------------------------------- | -------------------------------- |
| `CURDATE()` / `CURRENT_DATE()`      | 返回当前日期                     |
| `CURTIME()` / `CURRENT_TIME()`      | 返回当前时间                     |
| `NOW()`                             | 返回当前日期和时间               |
| `TIMESTAMPDIFF(part, date1, date2)` | 返回两个日期之间相差的指定单位值 |

```sql
SELECT CONCAT_WS('@', `name`,salary)
FROM employee;

SELECT CURDATE();

SELECT CURTIME();

SELECT TIMESTAMPDIFF(DAY,'2010-1-1 11:11:11','2010-1-2 11:11:12');

SELECT *,
TIMESTAMPDIFF(YEAR, birthday, CURDATE()) as age
from employee
ORDER BY age;
```

### 聚合函数

把**一堆多条数据**，算出**一个结果**的函数。

| 函数         | 作用                   |
| ------------ | ---------------------- |
| `AVG(col)`   | 求平均值               |
| `COUNT(col)` | 统计非 `null` 的记录数 |
| `COUNT(*)`   | 统计总记录数           |
| `MIN(col)`   | 求最小值               |
| `MAX(col)`   | 求最大值               |
| `SUM(col)`   | 求和                   |

```sql
SELECT AVG(salary) as `avg`, id FROM employee;

-- 查询员工数量
SELECT COUNT(id) FROM employee;

SELECT count(id) as 员工数量,
avg(salary) as 平均薪资,
sum(salary) as 总薪资,
min(salary) as 最小薪资
FROM employee;
```

## 分组

把**多条相同特征的数据**，归成**同一组**，合并在一起。
男生放一堆，女生放另一堆。

运行顺序

```sql
from
join ... on ...
where
group by
select
having
order by
limit
```

> 分组后，只能查询分组的列和聚合列

对于分组后的结果，如果要进行条件查询，需要使用**HAVING**关键词

```sql
-- 查询员工分布的居住地，以及每个居住地有多少名员工
-- 天府三街 3
SELECT location, count(id) as empnumber
FROM employee
GROUP BY location
HAVING empnumber>=40

-- 查询所有薪水在10000以上的员工的分布的居住地，然后仅得到聚集地大于30的结果
SELECT location, count(id) as empnumber
FROM employee
WHERE salary>=10000
GROUP BY location
HAVING count(id)>=30

-- 按照多字段分组 (只有地区，性别都一样的才算一组)
SELECT area,gender,COUNT(\*)
FROM user
GROUP BY area,gender;
```

## 练习

```sql
-- 1. 创建一张team表，记录足球队
-- 查询出对阵表

SELECT t1.name 主场, t2.name 客场
FROM team as t1, team as t2
WHERE t1.id != t2.id;

-- 2. 显示出所有员工的姓名、性别（使用男或女显示）、入职时间、薪水、所属部门（显示部门名称）、所属公司（显示公司名称）

SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id

-- 3. 查询腾讯和蚂蚁金服的所有员工姓名、性别、入职时间、部门名、公司名

SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id
WHERE c.`name` in ('腾讯科技', '蚂蚁金服')

-- 4. 查询渡一教学部的所有员工姓名、性别、入职时间、部门名、公司名

SELECT e.`name` 员工姓名,
case ismale when 1 then '男' else '女' end 性别,
e.joinDate 入职时间,
e.salary 薪水,
d.`name` 部门名称,
c.`name` 公司名称
FROM employee e
inner join department d on e.deptId = d.id
inner join company c on d.companyId = c.id
WHERE c.`name` like '%渡一%' AND d.`name` = '教学部';

-- 5. 列出所有公司员工居住的地址（要去掉重复）

select DISTINCT location from employee;
```
