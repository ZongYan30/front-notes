# md5加密
[md5加密](https://www.npmjs.com/package/md5) hash加密算法的一种，可以将任何一个字符串，加密成一个固定长度的字符串
* 单向加密：只能加密无法解密
* 同样的源字符串加密后得到的结果固定
* 有极小的概率两个不同的源字符串得到相同的结果
* 密码简单可以通过一些工具进行破解

## 使用
```sh
npm i md5 -S
```
```js
const secretedMsg = md5("message");
```

## 实践
```js
const Admin = require("../models/Admin.js");
const md5 = require("md5");

/* 新增 */
exports.addAdmin = async (adminObj) => {
	/* 法二:自动保存 */
	adminObj.loginPwd = md5(adminObj.loginPwd);//进行md5加密
	const ins = await Admin.create(adminObj); //创建得到实例
	return ins.toJSON();
}

// 登录
exports.login = async (loginId, loginPwd) => {
	loginPwd = md5(loginPwd);//登录的时候再加密一次

	const result = await Admin.findOne({
		where: { loginId, loginPwd }
	});

	if (result && result.loginId === loginId && result.loginPwd === loginPwd) {
		return result.toJSON();
	}

	return null;
}
```
