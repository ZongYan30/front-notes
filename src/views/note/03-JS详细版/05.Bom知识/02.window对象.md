
# window对象

## 自身方法

- open

打开一个新窗口

open("页面路径", "打开目标", "配置")

- alert
- confirm
- prompt

```js
const newWindow = window.open("https://www.baidu.com", "_blank"); //返回一个新的window对象

newWindow.resizeTo("1000px", "800px"); //resizeTo方法可以用在此处

newWindow.close(); //关闭当前窗口
```

## 对象属性

- reload方法：刷新当前页面

- document
  document.write
  在当前文档流中写入内容，如果当前文档流不存在，则新开一个文档流

- location：地址栏对象
  href属性：得到目前地址
  host属性：得到主机名
  port属性：得到端口号
  pathname属性：得到路径名
  search属性：得到查询字符串
  hash属性：得到哈希值
  protocol属性：得到协议
  hostname属性：得到主机名

- navigator：浏览器对象
  appName属性：得到浏览器名称
  appVersion属性：得到浏览器版本
  userAgent属性：得到浏览器的用户代理字符串

- history：历史记录
  go属性：跳转到指定位置
  back方法：上一页
  forword方法：下一页

- console
  log方法：打印对象的valueOf的返回值
  dir方法：打印对象结构
  tiem方法和timeEnd方法：用于计时
